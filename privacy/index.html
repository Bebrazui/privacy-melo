Политика конфиденциальности

Приложение «Melo» · Последнее обновление: 22 июня 2026 г.

Настоящая Политика описывает, какие данные обрабатывает мобильное приложение Melo (далее — «Приложение»), для чего и как вы можете ими управлять. Используя Приложение, вы соглашаетесь с условиями этой Политики.

Коротко: Приложение можно использовать полностью без регистрации и без передачи каких-либо данных — в локальном режиме. Данные обрабатываются только если вы сами включаете соответствующие функции (вход в аккаунт, синхронизация, «Карта музыки»). Рекламы, продажи данных и сторонней аналитики-слежки нет.

1. Кто обрабатывает данные

Оператором является разработчик Приложения. По вопросам конфиденциальности и удаления данных: namoknikisa@gmail.com.

2. Какие данные и зачем

  • Вход в аккаунт (по желанию). При входе через Google или по email/паролю обрабатываются ваш email и имя. Используются только для авторизации и привязки вашей библиотеки. Пароли в открытом виде не хранятся.
  • Синхронизация библиотеки (по желанию). Если вы вошли в аккаунт, ваши плейлисты и избранное (названия треков и ссылки на них) сохраняются на сервере, чтобы быть доступными на других устройствах. Локальные файлы с вашего устройства на сервер не передаются.
  • «Карта музыки» (по желанию). Когда вы оставляете трек на карте, сохраняются выбранные координаты и данные трека; этот «пин» становится виден другим пользователям. Геолокация запрашивается только в момент такого действия и только с вашего разрешения. Если вы не пользуетесь картой — местоположение не собирается.
  • Локальный режим. Плейлисты, избранное, история, скачанные и импортированные треки хранятся только на вашем устройстве и никуда не отправляются.

Приложение не использует рекламу, не ведёт рекламную слежку и не продаёт ваши данные.

3. Разрешения устройства

  • Интернет — воспроизведение и поиск музыки из открытых публичных источников.
  • Местоположение — только для функции «Карта музыки», когда вы сами оставляете трек на карте.
  • Уведомления — отображение управления плеером (название трека, пауза/перемотка) во время воспроизведения.
  • Доступ к файлам/папке — только для импорта вашей локальной музыки и сохранения скачанных треков, по вашему выбору.

4. Передача данных третьим лицам

Для работы перечисленных функций используются сторонние сервисы исключительно в технических целях:

  • Appwrite — облачный бэкенд (хранение аккаунта, синхронизация библиотеки, пины карты).
  • Google Sign-In — если вы выбираете вход через Google.

Сама музыка воспроизводится из открытых публичных источников через соединение вашего устройства; Приложение не размещает и не хранит аудиофайлы на своих серверах.

5. Хранение и удаление

Данные аккаунта и синхронизированной библиотеки хранятся, пока существует ваш аккаунт. Вы можете в любой момент:

  • выйти из аккаунта и продолжить пользоваться Приложением локально;
  • очистить локальные данные через настройки устройства (удаление данных приложения);
  • запросить удаление аккаунта и связанных данных, написав на namoknikisa@gmail.com.

6. Дети

Приложение не предназначено для детей младше возраста, установленного применимым законодательством, и целенаправленно не собирает их данные.

7. Изменения политики

Мы можем обновлять эту Политику. Актуальная версия всегда доступна по этому адресу; дата последнего обновления указана выше.

8. Контакты

По любым вопросам, связанным с конфиденциальностью: namoknikisa@gmail.com.
